Bootstrap Date Picker

The date picker allows you to open small popup containing all dates of a month. With the cursors on the left and right upper corner you are able to navigate between the different months. The picker is based on the most advanced date library moment.js. It offers full support of i18n as well as a complete set of date formatting capabilities. Additionally the date picker has the following properties, configurable via attributes:

  • fastScroll: if enabled allows to scroll through the months if the cursor is above the date picker.
  • monthLabelFormat: allows to format the date label on the top of the widget (currenlty ('Dezember - 2012')).
  • dayLabelFormat: allows to format the days label below the month label (currently ('Do.', 'Fr.', ...)).
  • dayFormat: allows to format the days in the different rows of the picker itself (currently ('31', '01', ...)).
